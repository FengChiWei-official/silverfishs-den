## qwq
For a [[Complete Binary Tree|CBT]] with `n` nodes, we know
1. `leaves` count `cl` is $cl = \lfloor n / 2 \rfloor$.
2. `Interal nodes` count `il` is $il = \lfloor n / 2 \rfloor + 1$.

So $il = cl + 1$.
