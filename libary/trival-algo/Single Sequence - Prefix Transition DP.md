## Define
It is focusing on Sequence. When you found You are handling Longest Increasing Subarray, your problem can be solve in a local-view, so just use [[Two Points Algorithm]].

Though it is a DP problem, you can solve with [[Greedy Algorithm]]::[[Improved-DP (Greedy)]].

|        | Time       | Space |
| ------ | ---------- | ----- |
| DP     | O(n^2)     | O(n)  |
| Greedy | O(n log n) | O(n)  |
