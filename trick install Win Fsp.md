**安装选项指南**

对于我们之前讨论的 **VirtioFS 共享**功能：

- **Core (核心)**：这是**必须安装**的组件。它包含了 VirtioFS 正常工作所需的基本 Windows 文件系统代理框架。
- **Developer (开发者)** 和 **Kernel Developer (内核开发者)**：这些选项通常是为希望编写自己的文件系统驱动程序的开发人员准备的，对于普通用户使用 VirtioFS 是**不需要的**。
- **FUSE for Cygwin**：如果您不使用 Cygwin 环境，则**不需要**安装此组件。
