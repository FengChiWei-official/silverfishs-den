## Definition

$y=x^t$, where $t \in \mathbb{R}$.  
_(Note: When $t$ is not an integer, we usually restrict the domain to $x > 0$.)_

## Intuition

Consider $t \in \mathbb{Q}$ (rational numbers), which can be expressed as a simplified fraction $t = \frac{a}{b}$ (where $\text{gcd}(a,b)=1$):

## Condition 1: $b$ is Odd

The function is defined for all $x \in \mathbb{R}$ (including negative $x$). The symmetry then depends on $a$:

- If $a$ is Even: It is an Even Function (symmetric about the y-axis).
    
    - _Example: $x^{2/3}$_
    
- If $a$ is Odd: It is an Odd Function (symmetric about the origin).
    
    - _Example: $x^{1/3}$_
    

## Condition 2: $b$ is Even 

The function is neither even nor odd because it is only defined for $x \geq 0$.

- Domain: Restricted to $[0, +\infty)$.
- Intuition: You cannot take an even root (like a square root) of a negative number in real analysis.
    
    - _Example: $x^{1/2} = \sqrt{x}$_
