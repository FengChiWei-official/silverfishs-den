## Text



Based on the **Triangle Inequality**, we have:
$$|a + b| \le |a| + |b|$$

Subtracting $|b|$ from both sides yields:
$$|a + b| - |b| \le |a|$$

Let $\alpha = a + b$ and $\beta = b$. Then $a = \alpha - \beta$. Substituting these into the inequality above, we obtain:
$$|\alpha| - |\beta| \le |\alpha - \beta| \quad \dots (1)$$
> [[Linear Coordinate Transformation]]


By symmetry, if we swap $\alpha$ and $\beta$, we have:
$$|\beta| - |\alpha| \le |\beta - \alpha|$$
Since $|\beta - \alpha| = |\alpha - \beta|$, this simplifies to:
$$-(|\alpha| - |\beta|) \le |\alpha - \beta| \quad \dots (2)$$

Combining $(1)$ and $(2)$, we arrive at:
$$-|\alpha - \beta| \le |\alpha| - |\beta| \le |\alpha - \beta|$$

Which is equivalent to:
$$\big| \, |\alpha| - |\beta| \, \big| \le |\alpha - \beta|$$
