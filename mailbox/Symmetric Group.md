## Definition
Let a set $\mathbf{M} = \{f \mid f: \mathbf{T} \to \mathbf{T}, \text{f is bijective} \}$, a set of all bijective maps of $\mathbf{T} \to \mathbf{T}$ from a **finite set** of n elements to itself.

>[!note]
>Let $\mathbf{T}$ to be $\{1, 2, 3\}$
>Valid f like $f := \{(1,2), (2,3), (3,1)\}$
>Not Valid f like $f := \{(1,1), (2,1), (3,1)\}$
### Element
There are $n!$ Permutations totally.
> $\mid \mathbf{S}_n \mid = n!$

Basic elements are :
- [[Transposition Permutation]] (1,2)
- [[Cyclic Permutation]](1,2,3,..., n)


### Notation
$$
S_n
