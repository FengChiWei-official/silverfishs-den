## Definition

A pair a,b of integers is [[Relatively Prime]] **if and only if** there are integers r and s such that $ra + sb = 1$.

## Proof
Lemma1: [[Relatively Prime|Define of Relatively Prime]]
Lemma2: [[Characterization of GCD via Linear Combinations]]

given  integer: $a, b$
Lemma1 -> $\gcd a, b = 1$
Lemma2 -> $\exists r, s \in \mathbb{Z}, \text { s.t. } ra + sb = \gcd a,b$
then,
$$\exists r, s \in \mathbb{Z}, \text { s.t. } ra + sb = 1$$
