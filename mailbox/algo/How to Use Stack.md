## Definition
1. When you try to **Read Top** or **Pop**, you should check if it is **empty**!
2. You can use [[Sentinel Node]] to avoid **Empty Check**.
3. Pay attention on your space! **Copying raw data and create stacks** in loop might run out your memory.
4. Match several cases, Use **Map**! that can unify you case checking.

	 check '[]', '{}', '<>', use:
	 map = {'[':']', ...}
