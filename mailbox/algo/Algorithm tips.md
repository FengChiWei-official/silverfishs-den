 **负数取模（Modulo of Negative Numbers）**
 **雷点**：C++ 中负数取模结果可能为负数（例如 -5 % 3 = -2）。在涉及减法的模运算中极易出错。
**防雷**：计算 (a - b) % mod 时，应写为 ((a - b) % mod + mod) % mod。
