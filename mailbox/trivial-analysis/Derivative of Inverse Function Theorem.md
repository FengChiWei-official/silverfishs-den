## Definition

$f^{\prime}(y) = x$
$\frac{d f^{\prime}(y)}{dx} = \frac{dx}{dx}$
$\frac{d f^{\prime}(y)}{dy} = \frac{1}{\frac{dy}{dx}}$

## Example

Given $\tan ^{\prime}x = \sec^2 x$,
$\arctan^{\prime} y = \frac{1}{sec^2 x}$
You should find `y` by yourself.

$\tan^2 x +\sec^2 x  = 1$
$y^2 + \sec^2 x = 1$

so $\arctan ^ {\prime} y = \frac{1}{1+y^2}$
