## Definition

When you are facing a Ratio that needs Symmetric, you need [[Chebyshev Polynomials]].

You are suppose to check weather $f(x) = f(x^{-1})$ stands. If so stands, you can let $f(x)$ to be $g(x + \frac{1}{x})$.
