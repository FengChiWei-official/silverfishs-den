### Docker 是什么？
docker功能上类似[[虚拟机]]。

### docker 概念
> 了解这些概念
- 模板： 
- 容器：只读的
- 文件系统挂载
- 指令: run, copy...
- docker compose：配置相关配置和环境


### Workflow
1. PULL 模板： linux/ubuntu
2. copy 源代码
3. 构建自己的模板
4. 写启动脚本和 docker_compse.yml
5. 运行
