## kinds
there are 3 kind of dynamic partitioning
- [[paging]]
- [[segmentation]]
- [[segmentation-with-paging]]

| kind                     | implement                          | pros              | cons                   |
| ------------------------ | ---------------------------------- | ----------------- | ---------------------- |
| paging                   | fix small pages based on sys       | low fragmentation |                        |
| segmentation             | dynamic segmentation based on task | locgically        | lots of small free mem |
| segmentation with paging | dynamic segmentation on fix pages  |                   |                        |
