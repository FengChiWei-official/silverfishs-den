## 使用指南

### 如何添加标签

将以下行添加到笔记的**最顶部**（在任何其他内容之前）：

```markdown
#tag1 #tag2/subtag #tag3

## 笔记内容
...
```

**示例：**

```markdown
#concept #math/algebra/abstract #entity/abstract

## 定义
...
```

### 标签规范

- **一级分类**（必选）：`#concept`, `#principle`, `#algo`, `#technique`, `#history`, `#theorem`, `#index` 等
- **二级领域**（推荐）：`#math/...`, `#computer-science/...`, `#physics/...`, `#ml/...`, `#general/...`
- **实体分类**（可选）：`#entity/abstract`, `#entity/concrete`, `#entity/virtual`, `#entity/physical`

详见 [[meta-tag]] 了解完整的标签系统定义。

### 推荐的标签组合示例

#### 数学概念
```markdown
#concept #math/algebra/abstract #entity/abstract
```

#### 算法或技术
```markdown
#technique #computer-science/os #entity/concrete
```

#### 物理定律
```markdown
#principle #physics/electromagnetism #entity/physical
```

#### 导航/索引
```markdown
#index #computer-science/dl
```
