## Text

1. 观察问题 ──> 是否在求一个最优解 X（最大/最小），或求反函数 X = f^-1(Y)？
                │
                └──> 是 ──> 2. 评估 X 的搜索空间大小 (Range)
                             │
                             ├──> 空间极小 (如 < 10^5) ──> 3. 是否多次查询？
                             │                             ├──> 是 ──> 预处理哈希表 [O(1) 查询]
                             │                             └──> 否 ──> 暴力线性扫描整个空间
                             │
                             └──> 空间较大 (如 10^5 ~ 10^18 或连续区间) ──> 4. 是否满足单调性？
                                                                          ├──> 是 ──> 二分搜索值域 (二分答案)
                                                                          └──> 否 ──> 寻求其他算法 (DP/贪心/回溯)
