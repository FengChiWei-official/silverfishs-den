## 争议条目标注

- `assistance` → `assistentia`：`assistentia`/`assistentia` 属于后期或中世纪构词，古典拉丁更常用 `auxilium` 或 `adiutorium` 表示“帮助”；若追求严格古典同源，应优先使用 `auxilium`。
- `assurance` → `assecuratio`：`assecuratio` 为较晚期的法律/行政名词，古典语料用 `securitas` 或相关表达更合适；`assecuratio` 可视为后期派生。
- `attendance` → `attentio`：`attentio` 原意偏向“注意”，用于“出席”时语义上并非一一对应；古典常用 `adesse`（动词）或 `adsistentia`（较不常见的名词）。
- `compliance` → `obsequium`：`obsequium` 与“顺从/服从”词源相关，但“comply”在形成上受后期法语/拉丁影响，二者并非完全同源。
- `conveyance` → `convectio`：`convectio` 来自 `convehere`（运送），为名词化用法，但古典语料中更常见动词形式，保留时注为“名词化派生”。
- `dependence` → `dependentia`：`dependentia` 为后期派生形式；古典更常以动词短语 `dependere (a)` 或 `subiectio` 等表达从属关系。
- `maintenance` → `manutentio`：`manutentio` 为中古拉丁/后期名词化形式，源自短语 `manu tenēre`（手 + 持有）并经由古法语/古法语派生（标注：OF/MID）。如需更接近古典，可改用动词短语 `manu tenēre` 或其他解释性表达。
- `existence` → `existentia`：`existentia` 多见于中世纪拉丁，古典常以 `esse`（动词）描述存在，或用其他构式表达名词概念。
- `reference` → `referentia`：`referentia` 较晚期出现；古典中常依赖 `referre`（动词）或具体说明性短语来表达“参考/引用”的概念。
- `resistance` → `resistentia`：此类名词在古典语料中并不普遍（多为后期派生），可用 `resistentia` 标注为派生形式并视需要替换为更古典的表达。
- `tolerance` → `tolerantia`：`tolerantia` 在晚期拉丁中出现，词源可接受但属后期名词化形式；如需严格古典风格，可用短语或其他表达。
- `vigilance` → `vigilantia`：古典中常见 `vigilantia`，请注意拼写，若需保持古典一致，改为 `vigilantia`。

(注：以上注释指出了表中若干条目为中世纪或后期派生形式而非严格古典拉丁，并给出可替代的古典表达或说明用法差异。如需，我可逐条引用古典文本例证并修订为更严格的古典形式。）
